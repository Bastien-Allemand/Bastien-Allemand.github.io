Moteur de Jeu (WIP)

Prototype de moteur d'application basé sur OpenGL et l'architecture ECS

Download ZIP du projet Lien GitHub (*si possible)

Description du projet

  • Implémentation d'une architecture système composante-entité (ECS) pour maintenir un code découplé et évolutif.
  • Gestion d'un pipeline de rendu OpenGL incluant la gestion de scènes, la lumière et l'initialisation de géométrie 3D.
  • Développement d'infrastructures communes (entrées utilisateurs, boucles de jeu, gestion de fenestration avec GLFW).

Compétences et Outils

  • C++ 17
  • Programmation Graphique (OpenGL moderne / GLSL)
  • Architecture logicielle performante (ECS)
  • Outils environnementaux (CMake, GLFW, GLAD)
