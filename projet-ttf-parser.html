Parseur TTF (C++)

Lecture binaire de fichiers TrueType et géométrie analytique

Download ZIP du projet Lien GitHub (*si possible)

Description du projet

  • Parseur de polices personnalisé qui extrait les contours des glyphes et les transforme en géométrie triangulée.
  • Gestion du parsing binaire directement depuis des fichiers `TrueType` sans librairie tierce complexe.
  • Travail centré sur la gestion de la disposition mémoire, l'optimisation en C++ et la géométrie algorithmique (polygones, courbes de Bézier).

Compétences et Outils

  • C++ (Standard Library, parsing de fichiers binaires)
  • Mathématiques appliquées (géométrie, algorithmes de triangulation)
  • Structures de données bas niveau
  • CMake / Git
